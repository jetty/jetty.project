Welcome to Jetty 11

The Jetty project is a 100% Java Servlet Container which supports asynchronous server and client implementations of the HTTP, Websocket and HTTP/2 protocols. The project is 100% Open Source and hosted by the Eclipse Foundation at https://www.eclipse.org/jetty/.
