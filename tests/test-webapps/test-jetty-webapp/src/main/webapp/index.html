Welcome to Jetty 9

This is the Test webapp for the Jetty 9 HTTP Server and Servlet Container. It is deployed in $JETTY_HOME/webapp.demo and configured by $JETTY_HOME/start.d/900-demo.ini

  • Servet: Hello World
  • Dump: Request, Session, Cookie
  • JSP: examples
  • Comet Chat: Long Polling, WebSocket (Jetty API), WebSocket (javax.websocket)
  • Authentication
  • Dispatcher Servlet
  • Request Rewrite Servlet
  • static content: tiny, small, medium, large, large gziped
